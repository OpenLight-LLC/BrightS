# 32位 i386 平台适配计划

## 概述
将 BrightS 内核适配为通用 i386 架构支持，同时保持对 Intel Atom N270 的特殊优化。

## 支持目标CPU
- ✅ Intel Atom N270 / N450 / D525
- ✅ Intel Pentium 4 / Pentium M / Core Duo
- ✅ AMD Athlon XP / Sempron
- ✅ VIA C3 / C7
- ✅ Geode LX / GX

## 最小系统要求
| 组件 | 最低要求 |
|---|---|
| CPU | Pentium Pro 级 (i686) 支持 PAE |
| 内存 | 64MB RAM |
| 存储 | IDE / SATA 硬盘 |

## 架构适配层级
```
kernel/                 # 架构无关代码 100% 复用
└── arch/
    ├── x86_64/         # 现有64位实现
    └── i386/           # 新32位实现
        ├── boot/       # 实模式引导代码
        ├── cpu/        # GDT, IDT, TSS
        ├── interrupts/ # 中断处理
        ├── paging/     # PAE分页
        ├── syscall/    # 系统调用入口
        └── include/    # 架构头文件
```
